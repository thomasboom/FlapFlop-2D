# FlipFlop 2D

A 2D arcade game built with HTML5 Canvas, CSS, and JavaScript.

## Features

- Smooth gameplay with continuous game loop
- Dynamic canvas-based graphics
- Simple click/tap controls for jumping
- Randomly generated pipes that move across the screen
- Collision detection between character and obstacles
- Real-time score tracking

## Getting Started

1. Clone the repository
2. Start the dev server:
   ```bash
   python -m http.server
   ```
3. Open `http://localhost:8000` in your web browser
